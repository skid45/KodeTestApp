# KodeTestApp

Проект сделан по [заданию от компании KODE](https://github.com/appKODE/trainee-test-android)

# Стэк
 - Kotlin 
 - Single Activity 
 - Retrofit 
 - XML 
 - Gson 
 - Coroutines
